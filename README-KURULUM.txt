ANILASLAKCI.COM YAYIN PAKETI

Sunucu ana dizinine şu yapıyı yükle:
- index.html
- robots.txt
- sitemap.xml
- assets/ klasörü

Domain bağlandıktan sonra yapılacaklar:
1. HTTPS/SSL etkinleştir.
2. www sürümünü ana domaine 301 yönlendir.
3. HTTP sürümünü HTTPS sürümüne 301 yönlendir.
4. Google Search Console alan adı mülkünü doğrula.
5. sitemap.xml gönder ve ana sayfa için dizine ekleme iste.
6. LinkedIn ve GitHub profil bağlantılarını anilaslakci.com olarak güncelle.
